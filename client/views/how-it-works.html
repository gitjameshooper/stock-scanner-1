How it Works

Main Steps

  • Authentication/Symbol Prep/Get Stock Data - Authorize use of TradeKing.com API
  • Stock Tests - Runs all stocks through tests
  • Buckets - Put stocks in buckets for best trade setups

Authentication/Symbol Prep/Get Stock Data

  • Uses oAuth to authorize
  • Makes JSON call to get public and secret keys/tokens
  • Makes JSON call to get all stock symbols
  • Formats symbols into a string and starts scanning
  • Makes a call to TradeKing's REST API every second(api limit 1 call/sec.) to get data then puts them through the tests.

Stock Tests

  • Using these data points(under the data fields section) tests are run to determine if the stock fits a certain criteria
  • Volume Test

    This a volume tier process where it makes sure the cumlative volume is above a certain threshold on the hour every hour.
    All stocks go through this test first. If it passes then it moves to the next.

  • ABCD Test

    This test consists of three smaller tests

  • VWAP Test

    Checks to see how far

  • If the stock passes a test then it gets added to that bucket.

Buckets

  • Once it makes through all the stocks it then applies the buckets of data to the view.(avg. 4 secs.)
  • Three Main Buckets
    • ABCD
    • VWAP
    • Speed

Using the App

  • Start/Stop Buttons - Make the app start/stop scanning the stocks
  • Green Bar - Scanner is working
  • Red Bar - Scanner has stopped
  • Yellow Bar - There is an error
  • Click Settings and you will have variables you can change to update your scanning paramaters
